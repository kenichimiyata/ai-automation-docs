# AI Automation Platform - Documentation

> Public documentation for the AI Automation Platform
> Maintained by [@kenichimiyata](https://github.com/kenichimiyata)

## Overview

An end-to-end automation platform connecting GitHub Issues with VS Code Copilot Chat via Supabase Realtime.

```
GitHub Issue
     |
     v
GitHub Actions
     |
     v
Supabase DB (Realtime)
     |
     v
VS Code Extension
     |
     v
Copilot Chat
```
